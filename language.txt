javascript - 브라우저 스크립트 프로그래밍 언어 , 스크립트 언어는 실행전에 컴파일하지 않기때문에 실행하면서 오류가 생길 수 있음
python - 인터프리터 스크립트 프로그래밍 언어
    pip : 파이썬 패키지 매니저
java - 객체지향형 컴파일러 언어, 컴파일 언어는 실행전에 컴파일링을 통해 문법오류를 잡아내기 때문에 실행되지도 않음.
    gradle, maven : java 패키지 매니저
nodejs - javascript를 브라우저 밖에서도 사용할 수 있게 해주는 프로그램
    npm - nodejs 패키지 매니저

react.js - nodejs frontend framework    
angular, vue - react와 같은 frontend framework 

express - nodejs backend web application server framework 
spring, spring boot - java의 backend web application server framework 
django, flask - python의 backend web application server framework 

nginx - web server
apache - web server
