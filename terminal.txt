//mac
ls : 현재 경로에 존재하는 파일/폴더 확인        
pwd : 현재 경로 위치 
cd 폴더이름 : 위치이동 
cd .. : 뒤로이동 
touch test.txt : 파일생성 
mkdir 폴더이름 : 폴더 생성 
cat : 파일내용확인 
rm test.txt : 파일 삭제 
rmdir 폴더이름 : 폴더 삭제 
rm -r 폴더이름 : 파일을 가지고 있는 폴더 삭제하기 
cp 복사할파일이름 복사할경로/복사파일이름 
mv 이동할파일 이동할경로 
mv 이름변경할파일 변경할이름

du -h 폴더이름 : 폴더 용량 보기
top -o PID : memory 사용 현황 보기
